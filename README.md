DreamQuill

An AI-powered storytelling platform that transforms user prompts into engaging, coherent, and genre-based stories using Generative AI and prompt engineering.

DreamQuill is a web application that enables users to create unique stories by simply describing their ideas. Whether it's romance, fantasy, mystery, sci-fi, or adventure, DreamQuill leverages Generative AI to generate meaningful narratives with logical flow, emotional depth, and creative storytelling.

📖 How It Works
Step 1

User signs in or creates an account.

Step 2

Step 3

DreamQuill analyzes the prompt and extracts:
